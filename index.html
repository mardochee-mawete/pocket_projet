Plateformes Outils

PLATEFORME

Utilisez efficacement les plateformes informatiques collaboratives pour travailler en équipe, partager des fichiers, communiquer et collaborer de manière productive

Slack

Un outil pour faciliter la communication au sein d'une entreprise, d'un groupe ou d'une structure afin de mener à bien les tâches au quotidien

ACCEDEZ

Asana

Une solution de gestion de tâches qui permet de planifier et organiser son travail dans un environnement très ludique et 100% collaboratif

ACCEDEZ

Google Classroom

Une plateforme d'apprentissage gratuite. Son but est de simplifier la création et la diffusion de cours et d'exercices de façon numérique

ACCEDEZ

Replit

Un environnement de développement intégré en ligne. Replit étant un logiciel en tant que service, il permet aux utilisateurs de créer des projets en ligne et d'écrire du code

ACCEDEZ

GitHub

Un service d'hébergement Open-Source, permettant aux développeurs de partager le code informatique de leurs projets afin de travailler dessus de façon collaborative

ACCEDEZ

OUTILS

Focus sur quelques outils front-end du développement web

figma

Un outil pour la conception de maquettes et de prototypes d'interface utilisateur. Accessible via un navigateur web, il permet une collaboration en temps réel entre les membres de l'équipe et un accès multi-plateforme

font Awesome

Une boîte à outils de polices et d'icônes basée sur CSS. L'utilisation directe des balises SVG dans votre code HTML va permettre d'optimiser la vitesse de chargement en limitant le nombre de requêtes HTTP

Google Fonts

Un outil gratuit qui permet de télécharger des polices d'écritures et des icônes. Ces éléments typographiques sont open source et peuvent donc être utilisées pour des projets personnels et commerciaux

validateur HTML W3C

Un outil libre créé par w3c qui vérifie que le code HTML d'une page Web est correctement formaté et respecte les normes Web actuelles. 3 types de validation disponibles : par Url, par chargement de fichier, par saisi direct

validateur CSS W3C

Le service de validation CSS du W3C est un logiciel libre créé par le W3C afin d'aider les développeurs Web à vérifier leurs feuilles de style en cascade (CSS)

freepik

Un site de banque d'images de haute qualité. La plateforme comprend des photographies, des illustrations, des images vectorielles et tout ce dont vous avez besoin pour vos projets créatifs
